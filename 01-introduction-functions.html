Introduction to functions:


Functions are sets of instructions that we can program one time and use every time we need them. Sometimes we will use functions created by other people, and sometimes we will have to create our own. “Using a function” is synonymous with calling or invoking a function.


functions.js 

                        function pintar_negro() {
                            elemento = document.querySelector("body");
                            elemento.style.backgroundColor = "black";
                        }
                    
                        pintar_negro();


Functions and Parameters:

Parameters allow us to input values into a function. The parameters go inside the parentheses when defining a function. The values passed when calling the function are formally known as arguments.

The term "arguments" and "parameters" are typically confused at first:

  • When defining a function → we use parameters
  • When executing a function → we pass arguments

parameters.js 
         
                        pintar = function(color){
                        elemento = document.querySelector("body")
                        elemento.style.backgroundColor = color
                        }
                        pintar("black");


Functions with Multiple Parameters:

Functions can have more than one parameter. To do this, we simply separate them with commas when defining the function. When calling the function, we must pass the arguments in the same order in which the parameters were defined.
