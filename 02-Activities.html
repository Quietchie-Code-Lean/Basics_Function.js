Activity 3: Increase and Decrease paragraph Font-size:

Creating a function with parameters to modify the DOM

Create a webpage with a paragraph containing the text “Font Size Control”, and add 4 buttons:

  • Increase by 5px
  • Decrease by 5px
  • Increase by 1px
  • Decrease by 1px
  1. Create a function called changeFontSize() with a default parameter value of 1.
  2. Inside the function, select the paragraph element and update its current font-size by adding the value passed as an argument.
  3. Call the function with the appropriate arguments for each button (for example: changeFontSize(5), changeFontSize(-5), changeFontSize(1), changeFontSize(-1)).
